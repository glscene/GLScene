Behaviours :

  • torque :
    • using the inertia behaviour for interface animations (Drakan-like effect)
    • accessing behaviours in the code
  • pathcontrol :
    • using the inertia movement behaviour to move objects along a path
  • FPSMovement :
    • using the FPSMovement behaviour and manager to simulate FPS-like movement inside freeForm maps
  • DCEDemo :
    • This is a basic use for the Dynamic Collision Engine (DCE). A TGLDCEManager, and several TGLBDCEBody behaviours on the objects that should interact.
